# OID_GEN_CO_RCV_CRC_ERROR

The OID_GEN_CO_RCV_CRC_ERROR OID specifies the number of PDUs received with cyclic redundancy check (CRC) errors.

## Requirements

| | |
| --- | --- |
| Version | Windows Vista and later |
| Header | Ntddndis.h (include Ndis.h) |
