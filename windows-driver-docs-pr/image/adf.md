# ADF element


The optional **ADF** element describes the capabilities of the automatic document feeder (ADF) that is attached to the scanner.

Usage
-----

``` syntax
<wscn:ADF>
  child elements
</wscn:ADF>
```

Attributes
----------

There are no attributes.
