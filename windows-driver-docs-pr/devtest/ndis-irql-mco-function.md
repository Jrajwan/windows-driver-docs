# Irql\_MCO\_Function rule (ndis)


The **Irql\_MCO\_Function** rule specifies that the NDIS MCO DDIs for miniport drivers must be called at correct IRQL levels.

|              |      |
|--------------|------|
| Driver model | NDIS |
