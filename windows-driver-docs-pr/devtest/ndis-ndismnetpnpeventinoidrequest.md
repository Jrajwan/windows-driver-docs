# NdisMNetPnPEventInOIDRequest rule (ndis)


This rule checks that [**NdisMNetPnPEvent**](https://msdn.microsoft.com/library/windows/hardware/ff563616) is not called in the context of an OID request.

|              |      |
|--------------|------|
| Driver model | NDIS |
